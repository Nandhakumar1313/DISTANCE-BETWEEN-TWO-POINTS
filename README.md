### PROGRAM:#Program to find the distance between two points.
```python

import math
m1=[4,2]
m2=[10,6]
d=math.sqrt(((m2[0]-m1[0])**2)+((m2[1]-m2[0])**2))
print("{:.2f}".format(d))
```
